Cara Pembelian

Melakukan transaksi di Anshel Store sangatlah mudah, instan, dan aman. Ikuti panduan langkah demi langkah berikut ini untuk melakukan Top Up Game atau memesan layanan kami.

1. Pilih Layanan atau Game

Pilih game yang ingin Anda Top Up melalui halaman beranda atau menu "Top Up". Jika Anda ingin memesan Jasa AI & Automation, silakan buka menu "Layanan".

2. Masukkan Data Akun Anda

Di halaman produk, Anda akan diminta untuk mengisi detail akun game Anda (contoh: User ID dan Server ID). Pastikan Anda mengisi data ini dengan teliti dan benar karena produk akan dikirimkan langsung ke ID tersebut.

3. Pilih Nominal dan Metode Pembayaran

  • Pilih jumlah Diamond, UC, atau produk lain yang ingin dibeli.
  • Pilih metode pembayaran yang paling nyaman untuk Anda (kami menerima QRIS, Transfer Bank, E-Wallet, dan Minimarket).
  • Masukkan nomor WhatsApp yang aktif untuk menerima bukti transaksi dan notifikasi status pesanan Anda.

4. Lakukan Pembayaran

Klik tombol Beli Sekarang. Anda akan diarahkan ke rincian pesanan. Selesaikan pembayaran sesuai dengan instruksi yang tertera di layar (termasuk kode unik jika ada) sebelum batas waktu pembayaran habis.

5. Pesanan Diproses Instan!

Setelah pembayaran Anda dikonfirmasi oleh sistem, pesanan akan langsung diproses secara otomatis. Produk digital akan masuk ke akun game Anda dalam hitungan detik (maksimal 10 menit jika server sedang padat).

Tips Melacak Pesanan

Anda dapat memantau status pesanan kapan saja melalui menu Cek Transaksi dengan memasukkan Nomor Invoice pesanan Anda.
