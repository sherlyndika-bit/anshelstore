Cara Pembelian dan Top-Up

Bertransaksi di Anshel Store dirancang untuk menjadi sangat mudah, cepat, dan 100% aman. Tidak perlu melakukan pendaftaran akun (meskipun login disarankan agar Anda dapat melacak riwayat pesanan). Ikuti panduan langkah demi langkah di bawah ini untuk memulai pengisian produk game Anda.

Langkah 1: Pilih Game atau Produk

  • Arahkan ke halaman utama Anshel Store.
  • Pilih game atau produk digital yang ingin Anda beli dari daftar katalog kami (misalnya: Mobile Legends, Free Fire, PUBG Mobile, Valorant, dll).
  • Klik pada logo atau nama game untuk masuk ke halaman detail pemesanan.

Langkah 2: Masukkan Data Akun (User ID)

Di halaman pemesanan, Anda akan diminta untuk memasukkan identitas akun game Anda. Mohon diperhatikan:

  • Masukkan User ID Anda dengan benar.
  • Jika game tersebut mensyaratkan Zone ID atau Server ID (seperti Mobile Legends), pastikan juga dimasukkan secara tepat.
  • Tip: Anda dapat menemukan User ID di halaman profil dalam game Anda. Terdapat tombol tanda tanya (?) di sebelah kolom input untuk melihat petunjuk bergambar.

Langkah 3: Pilih Nominal Top-Up

Setelah mengisi User ID, gulir ke bawah dan pilih nominal item, diamonds, atau voucher yang ingin Anda beli. Harga yang tertera adalah harga yang harus Anda bayar (sebelum biaya administrasi dari metode pembayaran, jika ada).

Langkah 4: Pilih Metode Pembayaran

Kami mendukung berbagai metode pembayaran di seluruh Indonesia untuk memudahkan Anda. Pilih metode yang paling nyaman bagi Anda:

  • E-Wallet: GoPay, OVO, Dana, ShopeePay, LinkAja.
  • QRIS: Bayar instan menggunakan aplikasi perbankan atau e-wallet apa pun yang mendukung scan kode QR.
  • Virtual Account (Transfer Bank): BCA, Mandiri, BNI, BRI, Permata.
  • Minimarket: Alfamart dan Indomaret.

Langkah 5: Masukkan Kontak & Konfirmasi Pesanan

  • Masukkan nomor WhatsApp (opsional, untuk menerima bukti transaksi langsung ke WhatsApp Anda) atau alamat Email Anda.
  • Klik tombol "Beli Sekarang" atau "Checkout".
  • Sebuah jendela pop-up akan muncul untuk memverifikasi nama akun (Nickname) Anda (khusus untuk game tertentu yang mendukung fitur cek nama otomatis). Periksa kembali dan pastikan Nickname sesuai dengan akun Anda.

Langkah 6: Lakukan Pembayaran

Selesaikan pembayaran sesuai dengan instruksi yang tertera di layar. Jika Anda memilih metode QRIS, segera scan kode QR yang muncul. Jika melalui Virtual Account, salin nomor rekening dan transfer sesuai nominal persis yang tertera hingga tiga digit terakhir (jika diwajibkan).

Langkah 7: Pesanan Berhasil!

Setelah pembayaran terkonfirmasi oleh sistem kami (biasanya dalam hitungan detik), produk akan otomatis dikirimkan ke akun game Anda secara instan. Anda akan langsung menerima Invoice Digital di layar serta notifikasi via WhatsApp/Email. Selamat bermain!

Penting: Jangan pernah membagikan bukti pembayaran atau nomor invoice kepada pihak yang tidak dikenal. Admin Anshel Store tidak pernah meminta Password akun game Anda.
